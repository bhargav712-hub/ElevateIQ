# Dashboard Endpoints

Here are the frontend route endpoints for each of the dashboards in the application:

## Student Portal
- `/student-dashboard`
- `/student-courses`
- `/student-courseDetails/:id`
- `/student-tests`
- `/student-assignments`
- `/student-recordings`
- `/student-exam-slots`
- `/student-payments`
- `/student-certificates`
- `/student-chat`

## Trainer Portal
- `/trainer-dashboard`
- `/trainer-courses`
- `/trainer-schedule`
- `/trainer-tests`
- `/trainer-assignments`
- `/trainer-recordings`
- `/trainer-students`

## Employee Portal
- `/employee-dashboard`
- `/employee-attendance`
- `/employee-leave`
- `/employee-meetings`
- `/employee-tickets`
- `/employee-announcements`

- `/admin-dashboard`
- `/admin-students`
- `/admin-trainers`
- `/admin-employees`
- `/admin-courses`
- `/admin-payments`
